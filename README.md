🎯 DeskAI — AI-Powered IT Helpdesk

DeskAI is a helpdesk web app that uses AI to analyze and manage support tickets. It helps organize tickets, filter them, and gives basic insights through a simple interface.

#Live

[https://deskai-j2vcj55da-sruchika1512-7581s-projects.vercel.app/](https://deskai-j2vcj55da-sruchika1512-7581s-projects.vercel.app/)

Demo: [demo@deskai.io](mailto:demo@deskai.io) / demo123


#What it does

* Uses AI to understand and analyze tickets
* Shows tickets instantly with a smooth UI
* Lets you filter and search easily
* Gives a basic dashboard for insights
* Includes login and signup system


Tech stack:-
React, Vite, Claude API, CSS, Vercel
